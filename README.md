# make-a-calculater
A basic calculator project built in Python.

## Description
This project provides a simple calculator that can perform addition, subtraction, multiplication, and division operations.

## How to Run
1. Clone the repository: `git clone https://github.com/faaz0saleem/make-a-calculater.git`
2. Navigate to the project directory: `cd make-a-calculater`
3. Run the calculator: `python calculator.py`

## Known edge cases
* Division by zero will raise a `ZeroDivisionError`.
* Non-numeric input will raise a `TypeError`.
